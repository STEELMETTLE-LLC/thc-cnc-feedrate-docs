# CNC Feedrate Baseline Docs (85A Compressed Air)

Documentation-only baseline material for mechanized CNC plasma startup tuning.

This repository intentionally excludes firmware and source code.

## Included Documents

- [CNC Feedrate Sheet.md](CNC%20Feedrate%20Sheet.md)
- [Baseline Feedrates 85A Compressed Air Tables.md](Baseline%20Feedrates%2085A%20Compressed%20Air%20Tables.md)

## Intended Use

- Use these values as startup baselines, not final production values.
- Tune feedrate first, then pierce timing, then height parameters.
- Validate on your own material, air system, and consumables.

## Practical Tuning Expectation

Expect to tune approximately 10% to 20% from baseline depending on machine condition and process variables.

## Safety and Quality Notes

- Confirm torch-to-work distance and pierce clearances before production cuts.
- Run test coupons before cutting production parts.
- Adjust for consumable wear, moisture control, and table dynamics.
