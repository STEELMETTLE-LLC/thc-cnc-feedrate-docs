# GitHub Release Text (85A Baseline)

## Release Title

85A CNC Compressed-Air Baseline Feedrates Published

## Release Notes

This release publishes a practical starting-point chart for mechanized CNC plasma tables in the 85A class using compressed air.

If you have struggled to find reliable startup values, this is meant to give you a fast path to first usable cuts.

What is included:

- Feedrate baseline from 20 GA through 1 in
- Pierce-time baseline in seconds
- Pierce-height and cut-height baseline
- Thickness-first tables by amp level

Important:

- These are startup values, not final locked production settings.
- Expect feedrate tuning in the range of about 10% to 20%.
- Pierce time, pierce height, and cut height may need adjustment based on material quality, air quality, table dynamics, and consumable condition.

Primary basis:

- Hypertherm mechanized chart family
- Lincoln Electric mechanized chart family

Use this as a baseline to reduce setup time, then tune with coupon cuts on your machine.
